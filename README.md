## 起動

```
docker-compose build
docker-compose up -d
docker-compose exec node sh
```

## 実行方法

```
※コンテナに入ってから
cd javascript
node sample.js
```

## 研修のプログラム作成方法

- javascript フォルダ配下に下記のようにフォルダ＆ファイルを作成

```
javascript
├ basic
│  ├ Level01
│  │  ├ Step01
│  │  └ Step02・・・
│  ├ Level02・・・
└ practice
   ├ Level01
   │  ├ Step01
   │  └ Step02・・・
   ├ Level02・・・
```

- node コマンドで js ファイルを実行して結果を確認する
